# Changelog

## [0.2.1] - 2026-05-10
### Added
- Session management with export/import
- Image generation endpoint
- Telegram bot integration
- GitHub Actions CI/CD workflow
- Oracle Cloud one-command installer
- Systemd service for production
- Portal landing page

## [0.2.0] - 2026-05-10
### Added
- Multi-LLM provider support with auto-fallback (4 providers)
- Composio MCP integration (30+ apps)
- GitHub API proxy
- Email sending via Composio
- Weather API (wttr.in)
- Built-in cron job scheduler
- Plugin system
- Channel support (Telegram, Discord scaffolding)
- Tool call parser with multiple format support
- Self-healing daemon (60s checks)
- Crontab healer (3 min checks)
- Boot script (one-command deploy)

## [0.1.0] - 2026-05-10
### Added
- FastAPI server with chat, WebSocket, REST API
- LLM integration (Vultr Inference API, OpenAI-compatible)
- Tool execution: shell, web search, memory, file ops, sysinfo
- Persistent memory with SQLite FTS5 full-text search
- Dark theme mobile-friendly web UI
- Docker + docker-compose ready
- Config management with YAML + env vars
